rc-slider

WAI-ARIA Slider Pattern

Basic — for/id label

Single thumb slider. Arrow keys adjust by one step; Page Up/Down by 10.

Volume: 40

Float Value Display — wrapping label

The display="float" mode floats the current value above the thumb. A wrapping <label> associates the label text with the input without any JavaScript.

Inline-Start Value Display — wrapping label

display="inline-start" renders the value before the track in the grid.

Direct aria-label on the input

When no visible label is needed, aria-label on the native input provides the accessible name directly — works pre-upgrade and with JS disabled.

Readonly

The thumb is visible and focusable but the value cannot be changed by the user.
