rc-textarea

A progressive enhancement component that adds line numbers, word wrap, range highlighting, Error Lens-style diagnostics, and pattern-based decoration to a native <textarea>.

Note: The host element requires explicit width + height (or use auto-grow for height). Without dimensions, the absolutely-positioned mirror and textarea have nothing to fill.
