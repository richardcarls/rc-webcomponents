rc-textarea-v2

WAI-ARIA contenteditable editor · Parchment document model · Plugin & decoration API

Active line + cursor state

New

Active line highlighted via --rc-textarea-v2-active-line-bg. Status bar reads api.selectionStart / api.selectionEnd via api.onCursorMove(). Blank lines inside a selection show a Monaco-style figure-space indicator.

Ln –, Col –  ·  Sel 0–0 (collapsed)

Pattern matching

Regex patterns with inline mark decorations via addPattern(). Type TODO, FIXME, or HACK anywhere.

Syntax highlighting — highlight.js

plugin.highlight() converts hljs HTML to mark decorations via api.decorationsFromHtml(). Theme CSS injected with api.adoptStyleSheet() — scoped to the shadow root, no leakage.

Diagnostics — line & mark decorations

LineDecoration with message renders an error-lens annotation via ::after (excluded from clipboard). Wavy underlines from MarkDecoration. Decorations are recomputed on each edit by plugin.update().

Widget decorations — inline color swatches

WidgetDecoration inserts a non-editable DOM element at a character offset. Edit the hex values and watch swatches update live.

Cursor-aware tooltip

New

Plugin uses api.onCursorMove() to track cursor changes, api.getWordAtCursor() to read the current word, and api.getCursorRect() to anchor a floating tooltip below the caret. Move the cursor over any keyword to see its definition.

Word wrap + auto-grow

word-wrap enables soft wrapping. auto-grow expands the editor to fit content with no scrollbar.
