rc-dialog

WAI-ARIA Dialog (Modal) Pattern

1. Basic modal dialog

Native showModal(), Escape closes. Events logged below.

Basic Dialog

Press Escape or click a button to close.

2. Movable dialog

Drag the titlebar to move. Tab to the titlebar; arrow keys move it (Shift = 10× step).

Movable Dialog

Drag the titlebar or use arrow keys (while titlebar is focused) to reposition.

3. Movable + resizable dialog

Drag any edge or corner to resize. Tab to the resize button (bottom-right); arrow keys resize (Shift = 10× step).

Resizable Dialog

Drag any edge or corner to resize.

4. Minimal dialog (no titlebar or footer)

A card-style dialog with an overlaid ✕. No structural titlebar or footer divs required.

Quick note

No structural titlebar or footer here — just content and an overlaid ✕ button.

Press Escape or ✕ to close.

5. Alert / Confirm dialog

Uses role="alertdialog" and aria-describedby for the alert message.

Delete item?
⚠️

This will permanently delete the item and cannot be undone.

6. Form dialog

Uses <form method="dialog">. Submitting closes the dialog and sets returnValue.

7. Close-guard (unsaved changes)

Listens for rc-dialog-request-close (cancelable). When the textarea has content, preventDefault() blocks the close.

8. Light-dismiss info panel

closed-by="any" (Chrome 134+/Safari 18.4+/Firefox 139+) plus light-dismiss JS fallback. Click the backdrop or press Escape to dismiss.
