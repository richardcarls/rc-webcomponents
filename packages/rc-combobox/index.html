rc-combobox

WAI-ARIA Combobox Pattern (editable)

Basic autocomplete

Type to filter the list. ↓ opens the popup and moves the virtual cursor. Enter or Tab selects. Escape clears the input and closes.

Allow-create

When allowcreate is set and the typed text has no exact match, a Create "X" option appears at the bottom of the list. The rc-combobox-create event is fired (cancelable).

Events will appear here…

Multi-select with chips

Select multiple items. Each selection becomes a chip in the input area. Backspace on empty input or ← at cursor start enters chip navigation mode.

Events will appear here…

Multi-select + Allow-create

Combine multi-select with allowcreate to build a fully dynamic tag input.

Form integration

The native <select slot="select"> is the form control. FormData reads its value on submit.

FormData will appear here after submit.
