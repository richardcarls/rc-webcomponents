Reference: WAI-ARIA Menubar Pattern

Application Menubar

Vertical Menubar

Status: Ready

Keyboard Controls

  • Left/Right Arrow: Navigate between menu buttons
  • Home/End: Jump to first/last menu button
  • Down Arrow or Enter/Space: Open menu
  • Escape: Close open menu
  • Tab: Exit menubar
  • When a menu is open, arrow navigation automatically opens adjacent menus
