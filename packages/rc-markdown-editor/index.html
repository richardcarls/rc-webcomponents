rc-markdown-editor

WAI-ARIA Toolbar Pattern

Basic

The default view is a contenteditable div rendering micromark HTML. Toolbar buttons apply formatting; Source toggles the markdown source editor. The hidden <textarea> in the light DOM carries the markdown value for form submission.

Rich editor + source toggle

Default view is rendered HTML. Click Source or press Ctrl+Shift+S to switch to the markdown source editor (powered by rc-textarea). Toolbar buttons reflect the formatting at the cursor via aria-pressed.

Events

rc-formatting-change fires when the cursor moves into or out of a formatted region. rc-mode-change fires when the source/rich toggle switches. Both bubble and are composed.

Keyboard shortcuts

All shortcuts are active when the editor is focused.

Ctrl+`
Inline code — <code> (rich) / `…` (source)
Ctrl+K
Link — prompts for URL (rich) / inserts [](url) (source)
