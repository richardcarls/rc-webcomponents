# rc-textarea

An enhanced textarea with line numbers, decorations, gutter support, and a plugin API while preserving the native `<textarea>`.

## Installation

`sh
npm install @rcarls/rc-textarea
`

`sh
yarn add @rcarls/rc-textarea
`

`js
import '@rcarls/rc-textarea/define';
`
