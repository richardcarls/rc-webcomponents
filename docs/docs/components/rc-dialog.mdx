# rc-dialog

A draggable, resizable wrapper around a native `<dialog>` that keeps browser dialog behavior intact.

<AtAGlance
  packageName="@rcarls/rc-dialog"
  tag="rc-dialog"
  native="Requires a native dialog child"
  state="Open state follows the native dialog"
  events={[
    'rc-dialog-open',
    'rc-dialog-toggle',
    'rc-dialog-request-close',
    'rc-dialog-cancel',
    'rc-dialog-close',
  ]}
  related={[{ label: 'Theme previews', href: '/guide/theme-previews' }]}
/>

## Installation

`sh
npm install @rcarls/rc-dialog
`

`sh
yarn add @rcarls/rc-dialog
`

`js
import '@rcarls/rc-dialog/define';
`
