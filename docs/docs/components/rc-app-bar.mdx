# rc-app-bar

A headless grid app bar with leading, title, exact-center, and trailing regions plus pinned, collapse, and hide scroll behaviors.

## Installation

`sh
npm install @rcarls/rc-app-bar
`

`sh
yarn add @rcarls/rc-app-bar
`

`js
import '@rcarls/rc-app-bar/define';
`
