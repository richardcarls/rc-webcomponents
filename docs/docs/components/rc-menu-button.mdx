# rc-menu-button

A button that opens an ARIA menu while preserving consumer-supplied trigger and menu content.

<AtAGlance
  packageName="@rcarls/rc-menu-button"

  state="Controlled or uncontrolled open state"

## Installation

`sh
npm install @rcarls/rc-menu-button
`

`sh
yarn add @rcarls/rc-menu-button
`

`js
import '@rcarls/rc-menu-button/define';
`
