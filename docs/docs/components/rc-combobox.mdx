# rc-combobox

An editable ARIA combobox with filtering, multi-select support, and optional allow-create behavior.

<AtAGlance
  packageName="@rcarls/rc-combobox"
  tag="rc-combobox"
  native="Requires a slotted native select"
  state="Controlled or uncontrolled text and selection"
  events={['rc-select-change', 'rc-combobox-create', 'rc-select-open', 'rc-select-close']}
  related={[{ label: 'Theme previews', href: '/guide/theme-previews' }]}
/>

## Installation

`sh
npm install @rcarls/rc-combobox
`

`sh
yarn add @rcarls/rc-combobox
`

`js
import '@rcarls/rc-combobox/define';
`
