# rc-toolbar

An ARIA toolbar that coordinates keyboard navigation across consumer-supplied controls.

## Installation

`sh
npm install @rcarls/rc-toolbar
`

`sh
yarn add @rcarls/rc-toolbar
`

`js
import '@rcarls/rc-toolbar/define';
`
