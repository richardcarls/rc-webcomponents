# rc-search-bar

Enhances a native `<input type="search">` with more consistent appearance and
behavior, and introduces convenience properties and debounced search events.

<AtAGlance
  packageName="@rcarls/rc-search-bar"

  state="Controlled or uncontrolled search value"
  events={['rc-search-bar-input', 'rc-search-bar-clear']}

## Installation

```sh
npm install @rcarls/rc-search-bar
```

```sh
yarn add @rcarls/rc-search-bar
```

```js
import '@rcarls/rc-search-bar/define';
```
