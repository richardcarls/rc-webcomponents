# rc-slider

A single-thumb slider backed by a native `<input type="range">` so labels, forms, and pre-upgrade interaction remain native.

<AtAGlance
  packageName="@rcarls/rc-slider"
  tag="rc-slider"
  native="Requires a native range input child"
  state="Controlled or uncontrolled numeric value"
  events={['rc-slider-input', 'rc-slider-change']}
  related={[{ label: 'Theme previews', href: '/guide/theme-previews' }]}
/>

## Installation

`sh
npm install @rcarls/rc-slider
`

`sh
yarn add @rcarls/rc-slider
`

`js
import '@rcarls/rc-slider/define';
`
