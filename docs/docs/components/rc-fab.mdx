# rc-fab

A Floating Action Button shell that positions a consumer-provided `<button>` in a fixed viewport
corner and supplies elevation styling. The button's own text content or `aria-label` is its
accessible name — no separate label property is needed.

<AtAGlance
  packageName="@rcarls/rc-fab"
  tag="rc-fab"
  native="Direct-child <button> element"
  state="Stateless button wrapper"
  events={[]}
  related={[{ label: 'Theme previews', href: '/guide/theme-previews' }]}
/>

## Installation

`sh
npm install @rcarls/rc-fab
`

`sh
yarn add @rcarls/rc-fab
`

`js
import '@rcarls/rc-fab/define';
`
