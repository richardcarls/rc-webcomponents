# rc-fab

A floating action button with regular and extended variants. Consumers own icons, labels, and placement.

## Installation

`sh
npm install @rcarls/rc-fab
`

`sh
yarn add @rcarls/rc-fab
`

`js
import '@rcarls/rc-fab/define';
`
