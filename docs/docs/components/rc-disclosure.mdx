# rc-disclosure

A disclosure wrapper around native `<details>` and `<summary>` with controlled/uncontrolled open support.

<AtAGlance
  packageName="@rcarls/rc-disclosure"

  state="Controlled or uncontrolled open state"
  events={['rc-disclosure-toggle']}
  related={[{ label: 'Theme previews', href: '/guide/theme-previews' }]}

## Installation

`sh
npm install @rcarls/rc-disclosure
`

`sh
yarn add @rcarls/rc-disclosure
`

`js
import '@rcarls/rc-disclosure/define';
`
