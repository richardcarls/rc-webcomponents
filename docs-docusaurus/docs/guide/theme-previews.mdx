# Theme previews

This spike imports `@rcarls/rc-theme-material/theme.css` globally, but Material styles activate only inside demo scopes with the `rc-theme-material` class.

The first pass does not port the full theme switcher. Instead, selected demos opt into Material so the spike can test whether Docusaurus/Infima styles interfere less with package-provided theme CSS.
